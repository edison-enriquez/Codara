# 3.2 Conversiones de tipo

---

## 3.2.1 ¿Por qué existen las conversiones?

Cuando un programa mezcla distintos tipos numéricos en una expresión (por ejemplo, un `int` con un `double`), C debe decidir con qué tipo operar. Este proceso se llama **conversión de tipo**.

Hay dos clases:
- **Implícita** — el compilador la hace automáticamente
- **Explícita** (*cast*) — tú la ordenas

---

## 3.2.2 Conversión implícita

Cuando mezclas tipos en una expresión, C promueve los valores al tipo "más capaz" siguiendo esta jerarquía:

```
char → short → int → long → long long → float → double
```

```c exec
#include <stdio.h>

int main(void)
{
    int    i = 7;
    double d = 3.5;
    double resultado;

    resultado = i + d;  // i se convierte automáticamente a double
    printf("7 + 3.5 = %f\n", resultado);

    // División entera vs flotante
    printf("7 / 2   = %d  (entera)\n",  7 / 2);
    printf("7 / 2.0 = %f  (flotante)\n", 7 / 2.0);

    return 0;
}
```

---

## 3.2.3 El peligro de la división entera

Un error clásico: dividir dos enteros cuando quieres un resultado decimal:

```c exec
#include <stdio.h>

int main(void)
{
    int piezas = 7;
    int grupos = 2;

    // Esto da 3, no 3.5 ← ¡error silencioso!
    printf("Resultado incorrecto: %d\n", piezas / grupos);

    // Solución: convertir al menos uno a double
    printf("Resultado correcto:   %f\n", (double) piezas / grupos);

    return 0;
}
```

---

## 3.2.4 El operador de cast explícito

Un **cast** es una conversión que tú escribes explícitamente con el tipo entre paréntesis:

```
(tipo) expresión
```

```c exec
#include <stdio.h>

int main(void)
{
    double x = 3.99;
    int    i;

    i = (int) x;   // trunca: 3 (NO redondea)
    printf("(int) 3.99  = %d\n", i);

    i = (int) -2.7;
    printf("(int) -2.7  = %d\n", i); // -2 (trunca hacia cero)

    float  f = 1234567;
    double d = (double) f;
    printf("float:  %.2f\n", f);
    printf("double: %.2f\n", d);

    return 0;
}
```

> **Recordar:** Los casts a `int` **truncan**, no redondean. `(int) 3.9` es `3`, no `4`.

---

## 3.2.5 Cast para forzar división flotante

El uso más común del cast es forzar aritmética de punto flotante con variables enteras:

```c exec
#include <stdio.h>

int main(void)
{
    int total = 10;
    int partes = 3;

    // Sin cast: división entera
    printf("Sin cast:    %d\n", total / partes);        // 3

    // Con cast: resultado flotante preciso
    printf("Con cast:    %f\n", (double) total / partes); // 3.333...

    // Porcentaje correcto
    int correctas = 7;
    int total_preguntas = 10;
    double porcentaje = (double) correctas / total_preguntas * 100;
    printf("Porcentaje: %.1f%%\n", porcentaje);

    return 0;
}
```

## 3.2.6 Conversión de `char` a `int` y viceversa

```c exec
#include <stdio.h>

int main(void)
{
    char letra = 'C';
    int  codigo;

    // char → int (implícito)
    codigo = letra;
    printf("'C' como entero: %d\n", codigo);

    // int → char (explícito para mayor claridad)
    char nueva = (char) (codigo + 1);
    printf("Siguiente letra: %c\n", nueva); // 'D'

    return 0;
}
```

---

## Resumen

| Concepto | Detalle |
|----------|---------|
| Conversión implícita | El compilador la hace automáticamente |
| Jerarquía | `char < int < long < float < double` |
| Cast explícito | `(tipo) expresión` |
| División entera | `7 / 2 = 3` — cuidado con enteros |
| Solución | `(double) 7 / 2 = 3.5` |
| Cast a `int` | Trunca (no redondea) |
