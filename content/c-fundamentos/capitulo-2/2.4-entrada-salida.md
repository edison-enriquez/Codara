## 2.4.1 La funciĂłn `printf()`

Ya conoces `printf()` de lecciones anteriores. En esta secciĂłn lo exploraremos en profundidad.

`printf()` toma una **cadena de formato** seguida de los valores a mostrar:

```

```

La cadena de formato puede contener:
- Texto normal (se imprime tal cual)
- **Especificadores de formato** (empiezan con `%`) â€” se reemplazan con los argumentos

---

## 2.4.2 Especificadores de formato

| Especificador | Tipo | Ejemplo |
|---------------|------|---------|
| `%d` o `%i` | Entero decimal (`int`) | `-42`, `100` |
| `%f` | Flotante decimal (`float`/`double`) | `3.140000` |
| `%e` | NotaciĂłn cientĂ­fica | `3.140000e+00` |
| `%g` | El mĂˇs corto entre `%f` y `%e` | `3.14` |
| `%c` | CarĂˇcter (`char`) | `A` |
| `%s` | Cadena de texto | `Hola` |
| `%x` | Entero en hexadecimal | `ff`, `1a3` |
| `%o` | Entero en octal | `47` |
| `%%` | El carĂˇcter `%` literal | `%` |

```c exec
#include <stdio.h>

int main(void)
{
    int    n = 255;
    float  f = 3.14159;
    char   c = 'C';

    printf("Decimal:     %d\n", n);
    printf("Hexadecimal: %x\n", n);  // ff
    printf("Octal:       %o\n", n);  // 377
    printf("Flotante:    %f\n", f);
    printf("CientĂ­fico:  %e\n", f);
    printf("AutomĂˇtico:  %g\n", f);
    printf("CarĂˇcter:    %c\n", c);

    return 0;
}
```

## 2.4.3 Anchura y precisiĂłn

Puedes controlar **cuĂˇntos caracteres** usa `printf()` para mostrar un valor:

### Anchura de campo

```
%10d    â†’ ocupa mĂ­nimo 10 caracteres, alineado a la derecha
%-10d   â†’ ocupa mĂ­nimo 10 caracteres, alineado a la izquierda
```

```c exec

{
    printf("[%10d]\n",  42);   // [        42]  derecha
    printf("[%-10d]\n", 42);   // [42        ]  izquierda
    printf("[%010d]\n", 42);   // [0000000042]  relleno con ceros

    return 0;
}
```

### PrecisiĂłn decimal

```
%.2f    â†’ 2 decimales
%.5f    â†’ 5 decimales
%8.2f   â†’ anchura 8, 2 decimales
```

```c exec

int main(void)
{
    float pi = 3.14159265;

    printf("%.2f\n",   pi);   // 3.14
    printf("%.5f\n",   pi);   // 3.14159
    printf("%10.3f\n", pi);   //      3.142
    printf("%-10.3f|\n", pi); // 3.142     |

    return 0;
}
```

## 2.4.4 Secuencias de escape en formato

| Secuencia | Efecto |
|-----------|--------|
| `\n` | Salto de lĂ­nea |

    printf("Ana GarcĂ­a\t25\tMadrid\n");
