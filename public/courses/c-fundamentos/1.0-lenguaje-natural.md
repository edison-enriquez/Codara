# 1.0 Lenguaje natural vs lenguaje de programación

## ¿Qué es el lenguaje?

Consideremos por un momento qué es el **lenguaje** en sí mismo. Podemos decir que un **lenguaje es una herramienta** para expresar y registrar los pensamientos humanos.

Los idiomas que usamos para comunicarnos con otras personas se llaman **lenguajes naturales**. Fueron creados a lo largo de muchos siglos y aún están sujetos a cambios.

Sin embargo, existen lenguajes cuya creación fue dictada por necesidades específicas, y su desarrollo está sujeto al control de amplios grupos de personas. Estos son los **lenguajes de programación**.

Un lenguaje de programación está definido por un conjunto de reglas rígidas, mucho más inflexibles que cualquier lenguaje natural. Por ejemplo:

- **Léxico**: qué símbolos (letras, dígitos, signos de puntuación) pueden usarse.
- **Sintaxis**: las formas apropiadas de combinar los símbolos.
- **Semántica**: el significado de cada instrucción.

> **Importante:** Cualquier programa que escribas debe ser correcto en estos tres aspectos. De lo contrario, no se ejecutará ni producirá resultados aceptables.

## El lenguaje de máquina

Una computadora carece de inteligencia. Solo responde a un **conjunto predeterminado de comandos conocidos**. Este conjunto se llama **lista de instrucciones** (*Instruction List*, IL).

La IL es el alfabeto de un lenguaje, comúnmente llamado **lenguaje de máquina**. Programar directamente en lenguaje de máquina es tedioso, consume mucho tiempo y es muy propenso a errores.

Estas circunstancias llevaron a la necesidad de algún tipo de **puente** entre el lenguaje humano y el lenguaje de la computadora: los **lenguajes de programación de alto nivel**.

## Compilación

La traducción de un lenguaje de alto nivel a lenguaje de máquina es realizada por un programa especializado llamado **compilador**.

```c
// Este es un ejemplo de código en C
// Nota: el código C NO se ejecuta en el navegador,
// pero lo aprenderás a compilar en tu computadora.
#include <stdio.h>

int main(void)
{
    printf("Hola desde lenguaje de alto nivel!\n");
    return 0;
}
```

El proceso es:

1. Escribes el **código fuente** (archivo `.c`)
2. El **compilador** lo traduce a lenguaje de máquina
3. El resultado es un **archivo ejecutable**

Si el compilador detecta errores de sintaxis o léxico, te los indicará. Los errores semánticos (lógica incorrecta) son tu responsabilidad.

## Algunas palabras sobre C

El lenguaje **C** fue creado a principios de los años 70 por [Dennis Ritchie](https://en.wikipedia.org/wiki/Dennis_Ritchie) en los Laboratorios Bell.

Es un **lenguaje de propósito general**, adecuado para casi cualquier proyecto de programación. Es especialmente útil para:

- Controladores (*drivers*)
- Sistemas embebidos
- Sistemas operativos (el kernel de Linux está escrito mayormente en C)
- Utilidades de sistema

Muchos lenguajes modernos (C++, Java, JavaScript, Python) tomaron ideas de C. Aprender C te dará una base muy sólida.
