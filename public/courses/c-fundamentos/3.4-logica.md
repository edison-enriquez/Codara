# 3.4 Lógica computacional

---

## 3.4.1 Operadores lógicos

Los operadores lógicos combinan varias condiciones en una sola expresión. Son esenciales para tomar decisiones more complejas en los programas.

| Operador | Nombre | Resultado |
|----------|--------|-----------|
| `&&` | AND (Y) | Verdadero solo si **ambas** condiciones son verdaderas |
| `\|\|` | OR (O) | Verdadero si **al menos una** condición es verdadera |
| `!` | NOT (No) | Invierte el valor lógico |

---

## 3.4.2 Operador AND (`&&`)

```c exec
#include <stdio.h>

int main(void)
{
    int contador = 5;
    int valor    = 100;

    // Ambas condiciones deben cumplirse
    if (contador > 0 && valor == 100) {
        printf("Condición AND cumplida.\n");
    }

    // Tabla de verdad AND
    printf("\nTabla AND:\n");
    printf("0 && 0 = %d\n", 0 && 0);
    printf("0 && 1 = %d\n", 0 && 1);
    printf("1 && 0 = %d\n", 1 && 0);
    printf("1 && 1 = %d\n", 1 && 1);

    return 0;
}
```

---

## 3.4.3 Operador OR (`||`)

```c exec
#include <stdio.h>

int main(void)
{
    int esAdmin   = 0;
    int tienePerm = 1;

    // Al menos una condición debe cumplirse
    if (esAdmin || tienePerm) {
        printf("Acceso concedido.\n");
    }

    // Tabla de verdad OR
    printf("\nTabla OR:\n");
    printf("0 || 0 = %d\n", 0 || 0);
    printf("0 || 1 = %d\n", 0 || 1);
    printf("1 || 0 = %d\n", 1 || 0);
    printf("1 || 1 = %d\n", 1 || 1);

    return 0;
}
```

---

## 3.4.4 Operador NOT (`!`)

```c exec
#include <stdio.h>

int main(void)
{
    int llueve = 0;

    if (!llueve) {
        printf("No llueve — sal a caminar.\n");
    }

    printf("!0 = %d\n", !0);  // 1 (falso se vuelve verdadero)
    printf("!1 = %d\n", !1);  // 0 (verdadero se vuelve falso)
    printf("!5 = %d\n", !5);  // 0 (cualquier número no-cero es verdadero)

    return 0;
}
```

## 3.4.5 Evaluación en cortocircuito

C evalúa las condiciones de izquierda a derecha y se **detiene** en cuanto conoce el resultado:

- En `a && b`: si `a` es falso, `b` nunca se evalúa (el AND ya es falso)
- En `a || b`: si `a` es verdadero, `b` nunca se evalúa (el OR ya es verdadero)

```c exec
#include <stdio.h>

int main(void)
{
    int x = 0;

    // Si x es 0, la segunda parte NO se evalúa (evita división por cero)
    if (x != 0 && (100 / x > 5)) {
        printf("Cociente mayor que 5\n");
    } else {
        printf("x es cero — la división se evitó con cortocircuito.\n");
    }

    return 0;
}
```

## 3.4.6 Combinando operadores lógicos

```c exec
#include <stdio.h>

int main(void)
{
    int edad    = 22;
    int ingreso = 3000;
    int credito = 750;

    // Solicitud de préstamo: mayor de 18, ingreso > 2000 o buen crédito
    if (edad >= 18 && (ingreso > 2000 || credito >= 700)) {
        printf("Préstamo aprobado.\n");
    } else {
        printf("Préstamo denegado.\n");
    }

    return 0;
}
```

> **Nota:** Usa paréntesis para aclarar el orden cuando mezclas `&&` y `||`. `&&` tiene mayor precedencia que `||`, pero los paréntesis hacen el código more legible.

## 3.4.7 Operadores a nivel de bits

Además de los operadores lógicos, C tiene operadores que actúan **bit a bit** sobre los valores enteros:

| Operador | Nombre | Ejemplo |
|----------|--------|---------|
| `&` | AND bit a bit | `5 & 3 = 1` |
| `\|` | OR bit a bit | `5 \| 3 = 7` |
| `~` | NOT bit a bit | `~5 = -6` |
| `^` | XOR bit a bit | `5 ^ 3 = 6` |
| `<<` | Desplazamiento izq. | `1 << 3 = 8` |
| `>>` | Desplazamiento der. | `8 >> 2 = 2` |

```c exec
#include <stdio.h>

int main(void)
{
    int a = 5;  // 0b00000101
    int b = 3;  // 0b00000011

    printf("a & b  = %d  (AND bit a bit)\n",  a & b);   // 1
    printf("a | b  = %d  (OR bit a bit)\n",   a | b);   // 7
    printf("a ^ b  = %d  (XOR bit a bit)\n",  a ^ b);   // 6
    printf("~a     = %d  (NOT bit a bit)\n",  ~a);       // -6
    printf("a << 1 = %d  (x2 con desplaz.)\n", a << 1); // 10
    printf("a >> 1 = %d  (÷2 con desplaz.)\n", a >> 1); // 2

    return 0;
}
```

---

## Resumen

| Operador | Tipo | Uso |
|----------|------|-----|
| `&&` | Lógico AND | Ambas condiciones deben ser verdaderas |
| `\|\|` | Lógico OR | Al menos una condición verdadera |
| `!` | Lógico NOT | Invierte el valor lógico |
| Cortocircuito | — | `&&` para en el primer `false`; `\|\|` para en el primer `true` |
| `&`, `\|`, `^`, `~` | Bit a bit | Operan sobre bits individuales |
| `<<`, `>>` | Desplazamiento | Multiplica/divide por potencias de 2 |
