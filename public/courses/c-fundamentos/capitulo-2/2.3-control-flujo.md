# 2.3 Control de flujo: la sentencia `if`

---

## 2.3.1 ¿Qué es el control de flujo?

Hasta ahora, los programas que hemos visto ejecutan instrucciones una tras otra en orden fijo. Pero en la vida real necesitamos tomar **decisiones**: *"si hace buen tiempo, salgo a caminar; si no, voy al cine"*.

El **control de flujo** permite al programa cambiar de camino según las circunstancias.

---

## 2.3.2 Operadores de comparación

Antes de tomar una decisión, la computadora necesita **comparar** valores. C tiene seis operadores de comparación:

| Operador | Significado | Ejemplo |
|----------|-------------|---------|
| `==` | Igual a | `a == b` |
| `!=` | Distinto de | `a != b` |
| `>` | Mayor que | `a > b` |
| `<` | Menor que | `a < b` |
| `>=` | Mayor o igual | `a >= b` |
| `<=` | Menor o igual | `a <= b` |

Una comparación devuelve:
- `1` (verdadero) si la condición se cumple
- `0` (falso) si no se cumple

```c exec
#include <stdio.h>

int main(void)
{
    int a = 5, b = 10;

    printf("a == b: %d\n", a == b);   // 0 (falso)
    printf("a != b: %d\n", a != b);   // 1 (verdadero)
    printf("a < b:  %d\n", a < b);    // 1 (verdadero)
    printf("a > b:  %d\n", a > b);    // 0 (falso)
    printf("a <= 5: %d\n", a <= 5);   // 1 (verdadero)

    return 0;
}
```

> ⚠️ **Error clásico:** Confundir `==` (comparación) con `=` (asignación). Son completamente diferentes.

---

## 2.3.3 La sentencia `if`

La sentencia `if` ejecuta una instrucción **solo si** la condición entre paréntesis es verdadera:

```
if (condición)
    instrucción;
```

```c exec
#include <stdio.h>

int main(void)
{
    int temperatura = 30;

    if (temperatura > 25)
        printf("Hace calor — puedes ir a la playa.\n");

    printf("Este mensaje siempre se muestra.\n");

    return 0;
}
```

Si `temperatura > 25` es verdad (lo es: 30 > 25), se imprime el primer mensaje. Si no, se saltaría.

---

## 2.3.4 Bloque de instrucciones con llaves

Si quieres ejecutar **más de una instrucción** cuando la condición es verdadera, usa llaves `{}`:

```c exec
#include <stdio.h>

int main(void)
{
    int lluvia = 1; // 1 = verdadero (hay lluvia)

    if (lluvia) {
        printf("Toma un paraguas.\n");
        printf("Ponte un abrigo.\n");
        printf("Lleva botas.\n");
    }

    printf("Que tengas buen viaje.\n");

    return 0;
}
```

> **Buena práctica:** Siempre usar llaves, incluso con una sola instrucción. Evita errores al añadir más líneas después.

## 2.3.5 Ejemplo: contador de ovejas negras

```c exec
#include <stdio.h>

int main(void)
{
    int ovejasBlancas = 10;
    int ovejasNegras  = 5;

    if (ovejasNegras == 2 * ovejasBlancas)
        printf("¡El doble de ovejas negras que blancas!\n");

    if (ovejasNegras < ovejasBlancas) {
        printf("Hay menos ovejas negras que blancas.\n");
        printf("Ovejas blancas: %d\n", ovejasBlancas);
        printf("Ovejas negras:  %d\n", ovejasNegras);
    }

    return 0;
}
```

## 2.3.6 Condiciones con variables `float`

```c exec
#include <stdio.h>

int main(void)
{
    float precio = 9.99;
    float presupuesto = 10.00;

    if (precio <= presupuesto)
        printf("¡Puedes comprarlo! Precio: %.2f\n", precio);

    if (precio > presupuesto)
        printf("Muy caro. Necesitas %.2f más.\n", precio - presupuesto);

    return 0;
}
```

> ⚠️ **Precaución con `float`:** Comparar igualdad exacta (`==`) con números de punto flotante puede dar resultados inesperados por errores de precisión. Prefiere comparar rangos (`>`, `<`) cuando trabajes con `float`.

---

## Resumen

| Concepto | Detalle |
|----------|---------|
| `==` | Comparación de igualdad (¡no confundir con `=`!) |
| `!=` | Desigualdad |
| `>`, `<`, `>=`, `<=` | Mayor, menor, mayor/igual, menor/igual |
| `if (cond)` | Ejecuta una instrucción si la condición es verdadera |
| `if (cond) { }` | Ejecuta un bloque de instrucciones |
| Verdadero / Falso | En C: `0` = falso, cualquier otro valor = verdadero |
