# 2.1 Operadores aritméticos

---

## 2.1.1 Operadores básicos

Los operadores aritméticos te permiten realizar cálculos con variables y valores numéricos.

| Operador | Nombre | Ejemplo |
|----------|--------|---------|
| `+` | Suma | `a + b` |
| `-` | Resta | `a - b` |
| `*` | Multiplicación | `a * b` |
| `/` | División | `a / b` |
| `%` | Módulo (resto) | `a % b` |

```c exec
#include <stdio.h>

int main(void)
{
    int i = 10, j = 12;
    float x = 1.25, y = 0.5;

    printf("Suma:           %d\n", i + j);
    printf("Resta:          %d\n", i - j);
    printf("Multiplicación: %d\n", i * j);
    printf("División entera:%d\n", i / j);
    printf("Módulo:         %d\n", i % j);

    printf("Float suma:     %f\n", x + y);
    printf("Float mult:     %f\n", x * y);
    return 0;
}
```

> **Nota:** El operador `%` (módulo) solo funciona con `int`, no con `float` ni `double`.

---

## 2.1.2 Precedencia y orden de operaciones

En matemáticas, la multiplicación y la división tienen mayor prioridad que la suma y la resta. En C es igual:

```
2 + 3 * 5  →  2 + 15  →  17   ✓
NO es: 5 * 5 = 25   ✗
```

```c exec
#include <stdio.h>

int main(void)
{
    int resultado;

    resultado = 2 + 3 * 5;
    printf("2 + 3 * 5 = %d\n", resultado); // 17

    resultado = (2 + 3) * 5;
    printf("(2 + 3) * 5 = %d\n", resultado); // 25

    return 0;
}
```

**Reglas de precedencia en C:**
1. `*`, `/`, `%` se evalúan antes que `+` y `-`
2. Operadores de igual prioridad se evalúan de **izquierda a derecha**
3. Los **paréntesis** siempre tienen la mayor prioridad

---

## 2.1.3 Ligadura a la izquierda

Cuando varios operadores tienen la misma prioridad, se evalúan de izquierda a derecha:

```c exec
#include <stdio.h>

int main(void)
{
    int resultado;

    resultado = 9 % 6 % 2;
    // = (9 % 6) % 2
    // = 3 % 2
    // = 1
    printf("9 %% 6 %% 2 = %d\n", resultado); // 1

    resultado = 8 / 4 / 2;
    // = (8 / 4) / 2
    // = 2 / 2
    // = 1
    printf("8 / 4 / 2 = %d\n", resultado); // 1

    return 0;
}
```

## 2.1.4 Operadores unarios

Los operadores unarios actúan sobre un **solo** operando:

```c exec
#include <stdio.h>

int main(void)
{
    int x = 5;

    printf("+x = %d\n", +x);  //  5 (no hace nada)
    printf("-x = %d\n", -x);  // -5 (cambia el signo)

    return 0;
}
```

## 2.1.5 Incremento y decremento

C tiene operadores especiales para sumar o restar 1 a una variable:

| Operador | Nombre | Efecto |
|----------|--------|--------|
| `i++` | Post-incremento | Usa el valor, luego suma 1 |
| `++i` | Pre-incremento | Suma 1, luego usa el valor |
| `i--` | Post-decremento | Usa el valor, luego resta 1 |
| `--i` | Pre-decremento | Resta 1, luego usa el valor |

```c exec
#include <stdio.h>

int main(void)
{
    int i = 10;

    printf("i = %d\n", i);

    i++;    // equivale a: i = i + 1
    printf("Después de i++: i = %d\n", i);

    i--;    // equivale a: i = i - 1
    printf("Después de i--: i = %d\n", i);

    printf("Valor de i++: %d (se usa 10, luego pasa a 11)\n", i++);
    printf("Ahora i = %d\n", i);

    printf("Valor de ++i: %d (primero pasa a 12, luego se usa)\n", ++i);
    printf("Ahora i = %d\n", i);

    return 0;
}
```

> **Precaución:** La posición del `++` importa cuando está dentro de una expresión más grande.

## 2.1.6 Operadores de asignación compuesta

Estos operadores abrevian la escritura de operaciones comunes:

| Forma larga | Forma corta |
|-------------|-------------|
| `x = x + 10` | `x += 10` |
| `x = x - 10` | `x -= 10` |
| `x = x * 10` | `x *= 10` |
| `x = x / 10` | `x /= 10` |
| `x = x % 10` | `x %= 10` |

```c exec
#include <stdio.h>

int main(void)
{
    int x = 100;

    x += 10;
    printf("Después de x += 10: %d\n", x); // 110

    x -= 5;
    printf("Después de x -= 5:  %d\n", x); // 105

    x *= 2;
    printf("Después de x *= 2:  %d\n", x); // 210

    x /= 3;
    printf("Después de x /= 3:  %d\n", x); // 70

    x %= 8;
    printf("Después de x %%= 8:  %d\n", x); // 6

    return 0;
}
```

## Resumen

| Concepto | Detalle |
|----------|---------|
| Operadores básicos | `+`, `-`, `*`, `/`, `%` |
| Precedencia | `*`, `/`, `%` antes que `+`, `-` |
| Paréntesis | Anulan la precedencia |
| Ligadura | Izquierda → derecha en igual prioridad |
| `i++` vs `++i` | Post vs pre-incremento |
| Compuestos | `+=`, `-=`, `*=`, `/=`, `%=` |
