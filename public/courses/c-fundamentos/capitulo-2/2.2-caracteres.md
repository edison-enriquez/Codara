# 2.2 Caracteres: el tipo `char`

---

## 2.2.1 El tipo `char`

Las computadoras procesan textos, pero por dentro solo entienden números. El tipo `char` es el puente entre los dos mundos.

Un `char` almacena **un solo carácter** (una letra, un número, un símbolo). En memoria ocupa exactamente **1 byte** y puede guardar valores entre -128 y 127 (con signo) o 0 y 255 (sin signo).

```c exec
#include <stdio.h>

int main(void)
{
    char letra = 'A';
    char digito = '5';
    char simbolo = '!';

    printf("Letra:   %c\n", letra);
    printf("Dígito:  %c\n", digito);
    printf("Símbolo: %c\n", simbolo);

    return 0;
}
```

> **Importante:** Los caracteres se escriben entre **comillas simples** (`'A'`), no dobles. Las comillas dobles son para cadenas de texto (`"HOLA"`).

---

## 2.2.2 El código ASCII

Cada carácter tiene asignado un número entero llamado **código ASCII** (American Standard Code for Information Interchange). Son 128 caracteres del 0 al 127.

Algunos valores clave:

| Carácter | Código ASCII |
|----------|-------------|
| `' '` (espacio) | 32 |
| `'0'` a `'9'` | 48 a 57 |
| `'A'` a `'Z'` | 65 a 90 |
| `'a'` a `'z'` | 97 a 122 |

```c exec
#include <stdio.h>

int main(void)
{
    char c = 'A';

    // %c muestra el carácter, %d muestra su código numérico
    printf("Carácter: %c   Código ASCII: %d\n", c, c);
    printf("Carácter: %c   Código ASCII: %d\n", 'a', 'a');
    printf("Carácter: %c   Código ASCII: %d\n", '0', '0');
    printf("Carácter: %c   Código ASCII: %d\n", ' ', ' ');

    return 0;
}
```

---

## 2.2.3 Aritmética con caracteres

Dado que `char` es un número por dentro, puedes hacer aritmética con él:

```c exec
#include <stdio.h>

int main(void)
{
    char mayus = 'A';
    char minus;

    // 'A' tiene código 65, 'a' tiene código 97
    // La diferencia siempre es 32
    minus = mayus + 32;

    printf("Mayúscula: %c (código %d)\n", mayus, mayus);
    printf("Minúscula: %c (código %d)\n", minus, minus);

    return 0;
}
```

Este truco funciona para **todas** las letras del alfabeto inglés. Por ejemplo:
- `'B'` (66) + 32 = `'b'` (98)
- `'Z'` (90) + 32 = `'z'` (122)

---

## 2.2.4 Recorrer letras con un bucle

Puedes usar la aritmética de caracteres para imprimir secuencias:

```c exec
#include <stdio.h>

int main(void)
{
    char c;

    printf("Mayúsculas: ");
    for (c = 'A'; c <= 'Z'; c++)
        printf("%c ", c);

    printf("\nMinúsculas: ");
    for (c = 'a'; c <= 'z'; c++)
        printf("%c ", c);

    printf("\nDígitos:    ");
    for (c = '0'; c <= '9'; c++)
        printf("%c ", c);

    printf("\n");
    return 0;
}
```

---

## 2.2.5 Secuencias de escape

Algunos caracteres especiales no se pueden escribir directamente y usan una secuencia con barra invertida:

| Secuencia | Significado | Código ASCII |
|-----------|-------------|-------------|
| `'\n'` | Nueva línea (enter) | 10 |
| `'\t'` | Tabulación | 9 |
| `'\\'` | Barra invertida literal `\` | 92 |
| `'\''` | Comilla simple literal `'` | 39 |
| `'\"'` | Comilla doble literal `"` | 34 |
| `'\0'` | Carácter nulo | 0 |

```c exec
#include <stdio.h>

int main(void)
{
    printf("Línea 1\nLínea 2\nLínea 3\n");
    printf("Col1\tCol2\tCol3\n");
    printf("Ruta: C:\\Users\\Nombre\n");
    printf("Dijo: \'hola\'\n");

    return 0;
}
```

## Resumen

| Concepto | Detalle |
|----------|---------|
| `char` | 1 byte — almacena un carácter |
| Literales | `'A'`, `'5'`, `'!'` (comillas simples) |
| `%c` | Especificador de formato para `char` |
| ASCII | Tabla de 128 caracteres (0-127) |
| `'A'` = 65 | `'a'` = 97 — diferencia de 32 |
| Aritmética | `'A' + 32` = `'a'` — conversión mayús/minús |
| Secuencias | `'\n'`, `'\t'`, `'\\'` — caracteres especiales |
