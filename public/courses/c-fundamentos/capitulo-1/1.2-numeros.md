# 1.2 Los números a través de la lente de una computadora

---

## 1.2.1 Los números y cómo los ven las computadoras

Las computadoras modernas trabajan con dos tipos de números:

- **Enteros** (*integers*): aquellos que carecen de parte fraccionaria.
- **Números de punto flotante** (*floating-point*): aquellos que contienen (o pueden contener) parte fraccionaria.

Esta distinción es muy importante y el límite entre estos dos tipos es **muy estricto**. La característica de un número que determina su tipo, rango y aplicación se llama **tipo**.

Conocemos dos tipos del lenguaje "C": el **tipo entero** (`int`) y el **tipo de punto flotante** (`float`).

---

## 1.2.2 Enteros en C

Un entero es simplemente una **cadena de dígitos**. No debes insertar ningún carácter que no sea dígito (sin comas, sin espacios). Por ejemplo, once millones ciento once mil ciento once se escribe:

```c
11111111
```

Los números negativos usan el signo `-` delante:

```c
-11111111
```

Los positivos no necesitan el signo `+`, pero puedes usarlo si quieres. Estas dos líneas describen el mismo número:

```c
+123
123
```

### Representaciones especiales

C permite escribir enteros en tres bases:

```c exec
#include <stdio.h>

int main(void)
{
    int decimal     = 255;    // base 10: el número 255
    int octal       = 0377;   // base  8: también es 255 (prefijo 0)
    int hexadecimal = 0xFF;   // base 16: también es 255 (prefijo 0x)

    printf("Decimal:     %d\n", decimal);
    printf("Octal:       %d\n", octal);
    printf("Hexadecimal: %d\n", hexadecimal);
    return 0;
}
```

> Los números octales solo pueden contener dígitos del 0 al 7. Los hexadecimales usan dígitos 0-9 y letras A-F.

---

## 1.2.3 Imprimir números

Para imprimir un número entero usamos `printf` con el especificador `%d`:

```c exec
#include <stdio.h>

int main(void)
{
    printf("%d\n", 42);
    printf("%d\n", -100);
    return 0;
}
```

Para imprimir un número de punto flotante usamos `%f`:

```c exec
#include <stdio.h>

int main(void)
{
    printf("%f\n", 3.14159);
    return 0;
}
```

En ambos casos debes incluir `#include <stdio.h>` al inicio del programa.

---

## 1.2.4 Una variable es variable

¿Cómo almacenamos los resultados de operaciones para usarlos después? Existen "contenedores" especiales llamados **variables**. El nombre sugiere que su contenido puede variar.

¿Qué tiene cada variable?

- un **nombre**
- un **tipo**
- un **valor**

---

## 1.2.5 Nombres de variables

Para dar nombre a una variable debes seguir reglas estrictas:

- el nombre debe estar compuesto de **letras latinas, dígitos y el carácter `_`** (guion bajo).
- el nombre debe **comenzar con una letra** (el guion bajo también cuenta como letra).
- las letras **mayúsculas y minúsculas son diferentes**: `Counter` y `counter` son dos variables distintas.

Las mismas restricciones aplican a los **nombres de funciones**.

**Nombres correctos:** `variable`, `i`, `t10`, `Exchange_Rate`, `counter`, `DaysToEnd`, `_`

**Nombres incorrectos:**
- `10t` — no comienza con una letra
- `Exchange Rate` — contiene un espacio
- `int` — es una palabra reservada del lenguaje

---

## 1.2.6 Tipos y declaración de variables

El **tipo** define qué valores puede almacenar la variable. Una declaración vincula un nombre con un tipo:

```c
int Counter;
```

También puedes declarar varias variables del mismo tipo en una línea:

```c
int variable1, account_balance, invoices;
```

Toda declaración termina con punto y coma.

---

## 1.2.7 Asignación de valores

Para darle un valor a una variable usas el **operador de asignación** `=`:

```c exec
#include <stdio.h>

int main(void)
{
    int Counter;
    Counter = 1;
    printf("Counter = %d\n", Counter);

    int result;
    result = 100 + 200;
    printf("result = %d\n", result);

    int x = 5;
    x = x + 1;   // x ahora vale 6
    printf("x = %d\n", x);

    return 0;
}
```

> En C el signo `=` no significa *"es igual a"*, sino **"asignar un valor"**.

También puedes inicializar la variable en la misma declaración:

```c
int edad = 25;
float precio = 9.99;
```

---

## 1.2.8 Palabras clave (*keywords*)

Se llaman **palabras reservadas** porque no debes usarlas como nombres de variables ni funciones. Algunos ejemplos:

| `int` | `float` | `void` | `return` | `if` |
|-------|---------|--------|----------|------|
| `else` | `while` | `for` | `do` | `switch` |
| `char` | `double` | `long` | `short` | `unsigned` |

Como el compilador de "C" es **sensible a mayúsculas y minúsculas**, puedes modificar estas palabras cambiando el caso de alguna letra, creando así un identificador válido:

```c
// ❌ Incorrecto — int es palabra reservada
// int int;

// ✅ Correcto — Int no es palabra reservada
int Int;
```

## 1.2.9 Comentarios en C

Los comentarios son notas dirigidas a los humanos, no al compilador. Sirven para **explicar** el código y son completamente ignorados durante la compilación.

Un comentario comienza con `/*` y termina con `*/`. Puede abarcar varias líneas:

```c
/* La variable Counter cuenta el número de ovejas en el prado */
int Counter;
```

```c
/*************************************************
 * Contando ovejas versión 1.1
 * Autor: Ronald Sleepyhead, 2024
 *************************************************/
```

También existe el comentario de una sola línea (estilo C99), usando `//`:

```c
int x = 5;  // esto es un comentario de una línea
```

> Los comentarios **anidados** (`/* ... /* ... */ ... */`) pueden causar problemas. Se recomienda evitarlos.

Los comentarios son útiles, pero no siempre son la mejor manera de hacer el código legible. A veces es mejor usar **nombres de variables descriptivos**:

```c exec
#include <stdio.h>

int main(void)
{
    /* 3 horas * 60 minutos * 60 segundos */
    int seconds_in_3_hours = 3 * 60 * 60;
    printf("Hay %d segundos en 3 horas.\n", seconds_in_3_hours);
    return 0;
}
```
