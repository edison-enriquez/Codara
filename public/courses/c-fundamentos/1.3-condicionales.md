# 1.3 Condicionales — Tomar decisiones en C

Un programa que siempre hace lo mismo tiene un valor limitado. La utilidad real aparece cuando el programa puede **tomar decisiones** en función de los datos.

## La instrucción `if`

```c exec
#include <stdio.h>

int main(void)
{
    int temperatura = 38;

    if (temperatura > 37) {
        printf("Tienes fiebre.\n");
    }

    printf("Consulta terminada.\n");
    return 0;
}
```

La condición entre paréntesis se evalúa como **verdadera** (`!= 0`) o **falsa** (`== 0`). En C no existe un tipo `bool` nativo (en C99+ sí, con `<stdbool.h>`), pero cualquier valor distinto de cero es verdadero.

## `if` / `else`

```c exec
#include <stdio.h>

int main(void)
{
    int nota = 65;

    if (nota >= 70) {
        printf("Aprobado ✓\n");
    } else {
        printf("Reprobado ✗\n");
    }

    return 0;
}
```

## Cadena `if` / `else if` / `else`

```c exec
#include <stdio.h>

int main(void)
{
    int hora = 14;

    if (hora < 12) {
        printf("Buenos días\n");
    } else if (hora < 20) {
        printf("Buenas tardes\n");
    } else {
        printf("Buenas noches\n");
    }

    return 0;
}
```

> 💡 Modifica el valor de `hora` (0–23) y vuelve a ejecutar para ver los distintos mensajes.

## Operadores de comparación

| Operador | Significado         |
|----------|---------------------|
| `==`     | Igual a             |
| `!=`     | Distinto de         |
| `<`      | Menor que           |
| `>`      | Mayor que           |
| `<=`     | Menor o igual que   |
| `>=`     | Mayor o igual que   |

> ⚠️ **Error frecuente:** `=` es **asignación**, `==` es **comparación**. Escribir `if (x = 5)` asigna 5 a `x` *y siempre* es verdadero.

## Operadores lógicos

```c exec
#include <stdio.h>

int main(void)
{
    int edad = 20;
    int tiene_carnet = 1;  /* 1 = true en C */

    if (edad >= 18 && tiene_carnet) {
        printf("Puede conducir.\n");
    }

    if (edad < 16 || !tiene_carnet) {
        printf("No puede conducir.\n");
    }

    return 0;
}
```

| Operador | Significado |
|----------|-------------|
| `&&`     | AND — ambas condiciones deben ser verdaderas |
| `\|\|`   | OR — al menos una debe ser verdadera |
| `!`      | NOT — invierte la condición |

## La instrucción `switch`

Cuando comparas una misma variable contra muchos valores, `switch` es más legible que una cadena de `if/else if`:

```c exec
#include <stdio.h>

int main(void)
{
    int dia = 3;

    switch (dia) {
        case 1: printf("Lunes\n");     break;
        case 2: printf("Martes\n");    break;
        case 3: printf("Miércoles\n"); break;
        case 4: printf("Jueves\n");    break;
        case 5: printf("Viernes\n");   break;
        default: printf("Fin de semana\n");
    }

    return 0;
}
```

> ⚠️ El `break` es **obligatorio** al final de cada `case`. Sin él, el flujo "cae" (*fall-through*) al siguiente case automáticamente.

## El operador ternario

Una forma compacta de expresar un `if/else` simple:

```c exec
#include <stdio.h>

int main(void)
{
    int x = 7;
    const char *tipo = (x % 2 == 0) ? "par" : "impar";
    printf("%d es %s\n", x, tipo);
    return 0;
}
```

La sintaxis es: `condicion ? valor_si_verdadero : valor_si_falso`
