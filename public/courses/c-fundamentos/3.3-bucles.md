# 3.3 Bucles

---

## 3.3.1 ¿Qué es un bucle?

Un **bucle** (o ciclo) permite repetir un bloque de código múltiples veces sin tener que escribirlo una y otra vez. En lugar de escribir 100 líneas de `printf`, escribes el código una vez y le dices al programa cuántas veces repetirlo.

C tiene tres tipos de bucles:
1. `while` — repite mientras la condición sea verdadera
2. `do-while` — igual, pero siempre ejecuta al menos una vez
3. `for` — bucle compacto con inicialización, condición y actualización

---

## 3.3.2 El bucle `while`

```
while (condición) {
    // se repite mientras la condición sea verdadera
}
```

La condición se verifica **antes** de cada iteración. Si desde el inicio es falsa, el cuerpo nunca se ejecuta.

```c exec
#include <stdio.h>

int main(void)
{
    int contador = 1;

    while (contador <= 5) {
        printf("Vuelta %d\n", contador);
        contador++;
    }

    printf("Bucle terminado.\n");
    return 0;
}
```

---

## 3.3.3 Bucle infinito (y cómo salir)

Si la condición nunca se vuelve falsa, el bucle no termina. Esto puede ser intencional:

```c exec
#include <stdio.h>

int main(void)
{
    int i = 0;

    while (1) {           // 1 siempre es verdadero
        printf("i = %d\n", i++);
        if (i >= 5)
            break;        // salir del bucle con break
    }

    printf("Salí del bucle.\n");
    return 0;
}
```

> **`break`** termina el bucle inmediatamente y continúa con el código después de él.

---

## 3.3.4 El bucle `do-while`

```
do {
    // se ejecuta al menos una vez
} while (condición);
```

La diferencia con `while`: la condición se verifica **después** de ejecutar el cuerpo. Por eso siempre se ejecuta al menos una vez.

```c exec
#include <stdio.h>

int main(void)
{
    int numero;

    do {
        printf("Escribe un número entre 1 y 10: ");
        scanf("%d", &numero);
    } while (numero < 1 || numero > 10);

    printf("Elegiste: %d\n", numero);
    return 0;
}
```

`do-while` es perfecto para **validar entradas del usuario**: siempre preguntas al menos una vez.

---

## 3.3.5 El bucle `for`

El `for` es el bucle más compacto. Reúne tres partes en una línea:

```
for (inicialización; condición; actualización) {
    // cuerpo
}
```

```c exec
#include <stdio.h>

int main(void)
{
    int i;

    // equivale al while(contador <= 5) de antes
    for (i = 1; i <= 5; i++) {
        printf("Vuelta %d\n", i);
    }

    return 0;
}
```

Las tres partes del `for`:
1. **Inicialización** (`i = 1`) — se ejecuta una sola vez al principio
2. **Condición** (`i <= 5`) — se verifica antes de cada iteración
3. **Actualización** (`i++`) — se ejecuta al final de cada iteración

---

## 3.3.6 Conteo hacia atrás y saltos

```c exec
#include <stdio.h>

int main(void)
{
    int i;

    // De 10 a 1
    printf("Cuenta regresiva:\n");
    for (i = 10; i >= 1; i--) {
        printf("%d ", i);
    }
    printf("\n¡Despegue!\n");

    // De 0 a 100 en saltos de 10
    printf("\nMultiplos de 10: ");
    for (i = 0; i <= 100; i += 10) {
        printf("%d ", i);
    }
    printf("\n");

    return 0;
}
```

---

## 3.3.7 La instrucción `continue`

`continue` salta a la siguiente iteración, omitiendo el resto del cuerpo:

```c exec
#include <stdio.h>

int main(void)
{
    int i;

    printf("Números del 1 al 10 (sin los múltiplos de 3):\n");
    for (i = 1; i <= 10; i++) {
        if (i % 3 == 0)
            continue;     // saltar múltiplos de 3
        printf("%d ", i);
    }
    printf("\n");

    return 0;
}
```

## 3.3.8 Bucles anidados

Puedes colocar un bucle dentro de otro para crear patrones:

```c exec
#include <stdio.h>

int main(void)
{
    int fila, col;

    // Tabla de multiplicar (1-5)
    printf("   ");
    for (col = 1; col <= 5; col++)
        printf("%4d", col);
    printf("\n   --------------------\n");

    for (fila = 1; fila <= 5; fila++) {
        printf("%2d |", fila);
        for (col = 1; col <= 5; col++)
            printf("%4d", fila * col);
        printf("\n");
    }

    return 0;
}
```

---

## Resumen

| Bucle | Condición | Casos de uso |
|-------|-----------|-------------|
| `while (cond) { }` | Antes de cada iteración | Cuando no sabes cuántas veces se repite |
| `do { } while (cond);` | Después de cada iteración | Cuando necesitas ejecutar al menos una vez |
| `for (init; cond; upd) { }` | Antes de cada iteración | Cuando sabes exactamente cuántas veces |
| `break` | — | Salir del bucle inmediatamente |
| `continue` | — | Saltar al inicio de la siguiente iteración |
