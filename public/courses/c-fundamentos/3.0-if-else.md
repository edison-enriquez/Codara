# 3.0 La sentencia `if-else`

---

## 3.0.1 Dos caminos: `if` y `else`

En la lección anterior aprendiste a usar `if` para ejecutar código cuando una condición es verdadera. Pero ¿qué pasa cuando la condición es **falsa**?

La cláusula `else` ejecuta un bloque alternativo cuando la condición del `if` no se cumple:

```
if (condición) {
    // se ejecuta si la condición es VERDADERA
} else {
    // se ejecuta si la condición es FALSA
}
```

```c exec
#include <stdio.h>

int main(void)
{
    int buenTiempo = 0; // 0 = falso, 1 = verdadero

    if (buenTiempo) {
        printf("Salir a caminar.\n");
    } else {
        printf("Ir al teatro.\n");
    }

    printf("Almorzar.\n"); // esto siempre se ejecuta

    return 0;
}
```

Prueba a cambiar el valor de `buenTiempo` a `1` para ver el otro camino.

---

## 3.0.2 Cadena `else if`

Cuando necesitas más de dos opciones, encadena varias condiciones:

```c exec
#include <stdio.h>

int main(void)
{
    int nota = 75;

    if (nota >= 90) {
        printf("Calificación: A (Excelente)\n");
    } else if (nota >= 80) {
        printf("Calificación: B (Bien)\n");
    } else if (nota >= 70) {
        printf("Calificación: C (Aprobado)\n");
    } else if (nota >= 60) {
        printf("Calificación: D (Suficiente)\n");
    } else {
        printf("Calificación: F (Reprobado)\n");
    }

    return 0;
}
```

Solo se ejecuta **uno** de los bloques: el primero cuya condición sea verdadera.

---

## 3.0.3 `if-else` anidados

Puedes colocar un `if` dentro de otro:

```c exec
#include <stdio.h>

int main(void)
{
    int edad = 20;
    int tieneId = 1;

    if (edad >= 18) {
        if (tieneId) {
            printf("Acceso permitido.\n");
        } else {
            printf("Necesitas presentar identificación.\n");
        }
    } else {
        printf("Acceso denegado: menor de edad.\n");
    }

    return 0;
}
```

---

## 3.0.4 El problema del `else` huérfano

Cuando anidas `if` sin llaves, el `else` siempre pertenece al `if` **más cercano**:

```c exec
#include <stdio.h>

int main(void)
{
    int x = 10;

    // ¿A qué if pertenece el else?
    if (x > 0)
        if (x > 5)
            printf("x es mayor que 5\n");
        else
            printf("¿Esto es para x > 0 o x > 5?\n"); // pertenece al if(x > 5)

    return 0;
}
```

Para evitar confusión, **siempre usa llaves**:

```c exec
#include <stdio.h>

int main(void)
{
    int x = 10;

    if (x > 0) {
        if (x > 5) {
            printf("x es mayor que 5\n");
        }
    } else {
        printf("x es cero o negativo\n"); // ahora queda claro
    }

    return 0;
}
```

## 3.0.5 Ejemplo: clasificador de temperaturas

```c exec
#include <stdio.h>

int main(void)
{
    float temp = 22.5;

    printf("Temperatura: %.1f°C\n", temp);

    if (temp < 0) {
        printf("Estado: Bajo cero — ¡Hielo!\n");
    } else if (temp < 10) {
        printf("Estado: Muy frío\n");
    } else if (temp < 20) {
        printf("Estado: Fresco\n");
    } else if (temp < 30) {
        printf("Estado: Agradable\n");
    } else if (temp < 40) {
        printf("Estado: Caluroso\n");
    } else {
        printf("Estado: Muy caluroso — ¡Peligro!\n");
    }

    return 0;
}
```

---

## Resumen

| Estructura | Uso |
|------------|-----|
| `if (cond) { }` | Ejecuta si la condición es verdadera |
| `if (cond) { } else { }` | Un camino u otro, nunca ambos |
| `else if (cond) { }` | Múltiples condiciones encadenadas |
| `if` anidado | Un `if` dentro de otro |
| Llaves `{}` | Siempre recomendadas para evitar ambigüedad |
