# 1.2 Los números a través de la lente de una computadora

## Dos tipos de números

Las computadoras modernas trabajan con dos tipos de números:

| Tipo         | Nombre en C   | Ejemplo          |
|--------------|---------------|------------------|
| Enteros      | `int`         | `42`, `-7`, `0`  |
| Punto flotante | `float` / `double` | `3.14`, `-0.5` |

Esta distinción es **muy estricta** en C. No son lo mismo `5` (entero) y `5.0` (flotante), aunque en matemáticas representen el mismo valor.

## Enteros en C

Un entero es simplemente una **cadena de dígitos**. No debes insertar espacios ni comas:

```c
// ✅ Correcto
int n = 11111111;

// ❌ Incorrecto (error de compilación)
// int n = 11,111,111;
```

Los enteros negativos usan el signo `-` delante:

```c
int temperatura = -15;
int deuda = -500;
```

### Representaciones especiales

C admite tres bases numéricas:

```c
int decimal     = 255;     // base 10: el número 255
int octal       = 0377;    // base 8:  también es 255 (prefijo 0)
int hexadecimal = 0xFF;    // base 16: también es 255 (prefijo 0x)
```

## Imprimir números

Para imprimir integers usamos `printf` con el especificador `%d`:

```c exec
#include <stdio.h>

int main(void)
{
    int edad = 25;
    printf("Tengo %d años.\n", edad);
    return 0;
}
```

Para floats usamos `%f`:

```c exec
#include <stdio.h>

int main(void)
{
    float pi = 3.14159;
    printf("Pi es aproximadamente %f\n", pi);
    return 0;
}
```

## Operaciones aritméticas

C soporta las operaciones básicas:

```c exec
#include <stdio.h>

int main(void)
{
    int a = 10, b = 3;

    printf("Suma:           %d\n", a + b);
    printf("Resta:          %d\n", a - b);
    printf("Multiplicación: %d\n", a * b);
    printf("División:       %d\n", a / b);  // 3 (¡división entera!)
    printf("Módulo:         %d\n", a % b);  // 1 (resto)

    return 0;
}
```

> ⚠️ **¡Atención!** La división entre enteros produce un entero. `10 / 3` en C da `3`, no `3.333...`. El resto se pierde.

Para una división exacta, usa `float` o `double`:

```c
float resultado = 10.0 / 3.0;  // 3.333333
```

## Desbordamiento (overflow)

Los enteros tienen un límite de tamaño. En la mayoría de sistemas, `int` ocupa 4 bytes y puede almacenar valores entre aproximadamente **-2.1 mil millones** y **+2.1 mil millones**.

Si superas ese límite, ocurre un **desbordamiento** (*overflow*):

```c exec
#include <stdio.h>

int main(void)
{
    int maximo = 2147483647;   // el valor máximo de int
    printf("%d\n", maximo);       // 2147483647
    printf("%d\n", maximo + 1);   // -2147483648 ← desbordamiento!
    return 0;
}
```

---

> **Regla práctica:** Usa `int` para números enteros cotidianos, `double` (más preciso que `float`) cuando necesites decimales, y `long int` o `long long int` cuando necesites enteros muy grandes.
